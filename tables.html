<caption> = immediately after the table tag, only 1 allowed
<thead> = the header content in a table, used after <table>, or after <caption> if it's there
<tbody> = specifies the body of a table
<tfooter> = specifies the footer content in a table
<tr> = table row; made up from <th> and <td> elements
<th> = header cell in a <tr>, holds header data, positioned in the center by default
<td> = plain cells for data in a <tr>, positioned to the left by default
